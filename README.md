# Wangdi Portfolio V7

Professional portfolio implementation for Wangdi, built on a reworked Gatsby `v4` baseline.

## Scope

- Public-facing professional portfolio
- Experience timeline
- Case studies
- Publications and manuscripts
- Downloadable CV

## Local Development

1. Install dependencies
   ```sh
   yarn
   ```
2. Start the development server
   ```sh
   yarn develop
   ```
3. Build for production
   ```sh
   yarn build
   ```

## Notes

- Update `siteUrl` in [gatsby-config.js](/D:/Portfolio/v7/gatsby-config.js:1) before deployment.
- Add final social profile links in [src/config.js](/D:/Portfolio/v7/src/config.js:1) when the LinkedIn and ResearchGate URLs are available.
